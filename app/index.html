About Us
We are a small team of designers passionate about great UX and nice UI. Our mission is to make software easier and more pleasurable to use. We passionate about great UX and love complex web interfaces and want to make sure people have a great time using them.
Research
Build User Personas, Create environment profiles, define key user journeys, create User Stories. Define the golas of the business and users.
Create UX
Set key performance indicators, design the interaction.
Create UI
Creating Wireframes. UI, visual design and create a UI style guide.
Validate designs
Analyze how users use the product. Can they get the goal that customer want them to?
Our Works

UI and UX for Aquapoint scheduling system

Our job was to redesign large SaaS system for pool owners, that gives them full remote control over different pools in different locations. Our job was to redesign a core functionality of a system - Scheduling part for pool owners to manage tasks for pool staff and a profile management system.

UI and UX for setup wizard for PBX phone system

Our job was to redesign of a cloud PBX installation wizard. This is difficult and important step to get access to a telecommunication system. Therefore it should be intuitive and easy to use. FOr us it was challenge to create a user-friendly interface for a user that has not previously used a telecommunication system.
